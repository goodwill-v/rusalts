# knowledge_base

Эта папка — **каноническая База знаний** АЛТ‑эксперт в структурированном виде.

Формат статей: Markdown + YAML‑шапка (см. `docs/KNOWLEDGE_BASE.md`).

Ключевые файлы:
- `articles/` — атомарные статьи по секциям
- `triggers.json` — триггеры для выдачи шаблонов (шаблоны не хранятся в БЗ)
- `changelog/` — журнал изменений БЗ (MD + JSONL)
