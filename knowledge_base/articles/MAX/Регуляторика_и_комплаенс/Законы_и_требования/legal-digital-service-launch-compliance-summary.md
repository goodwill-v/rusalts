Сводная памятка проекта **АЛТ** перед запуском публичного цифрового сервиса. Детали — в профильных статьях раздела `MAX/Регуляторика_и_комплаенс`.

## Единый чек-лист перед запуском

```
ПЕРЕД ЗАПУСКОМ ЧАТ-БОТА / СООБЩЕСТВА / СЕРВИСА:

[ ] 152‑ФЗ: Персональные данные
    [ ] Определены цели и перечень собираемых данных
    [ ] Опубликована Политика конфиденциальности
    [ ] Реализован механизм получения согласия
    [ ] Обеспечена локализация баз данных в РФ
    [ ] Назначен ответственный за обработку ПДн

[ ] 168‑ФЗ: Русский язык в публичных пространствах
    [ ] Аудит контента на иностранные слова
    [ ] Правила оформления с пояснениями
    [ ] Обновлены шаблоны ответов и интерфейс
    [ ] Обучены авторы контента

[ ] Региональная доступность и реестры
    [ ] Проверка домена и сервисов по процедуре компании
    [ ] Контент сверен с ограничениями законодательства РФ
    [ ] Регламент реагирования на блокировки
    [ ] Резервные каналы коммуникации

[ ] Документация и роли
    [ ] Шаблоны согласий, политик, уведомлений
    [ ] Ответственные за ПДн, контент, мониторинг реестров
    [ ] Журнал инцидентов и обращений

[ ] Мониторинг
    [ ] Отслеживание изменений законодательства
    [ ] Регулярный аудит контента и процессов
```

## Периодичность аудита (рекомендация)

| Область | Периодичность | Роль |
| ------- | ------------- | ---- |
| Контент на соответствие требованиям к русскому языку в публичных пространствах | Ежеквартально | Контент / юрист |
| Политика ПДн | При изменении обработки, минимум раз в год | Ответственный за ПДн |
| Реестры РКН | Еженедельно + автоматизация | Техспециалист / юрист |
| Обучение команды | При изменении законов, минимум раз в год | Руководитель продукта |
| Учения по инцидентам доступности | Раз в полгода | ИБ / техлид |
