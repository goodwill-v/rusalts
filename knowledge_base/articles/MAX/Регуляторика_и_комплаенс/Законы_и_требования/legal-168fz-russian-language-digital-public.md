Федеральный закон от 01.06.2023 № 168‑ФЗ вносит изменения в Закон РФ «О языках народов Российской Федерации» и отдельные акты. Ниже — **практическая выжимка** для сайтов, приложений, сообществ в мессенджерах (включая MAX), ботов и рассылок. Точные дефиниции «публичного пространства» и объём требований берите из **актуальной редакции** закона и разъяснений уполномоченных органов.

## Ключевые положения (ориентир)

| Фрагмент законодательной конструкции | Содержание (кратко) | Практическое значение |
| ------------------------------------ | ------------------- | --------------------- |
| Изменения в ст. 3 Закона о языках | Публичные пространства | В числе прочего: сайты, приложения, мессенджеры для распространения информации широкому кругу лиц |
| Изменения в ст. 4 | Русский как основной | В публичных пространствах на территории РФ русский — основной; иностранные слова — с пояснением на русском, если нет исключения |
| Изменения в ст. 5 | Исключения | Собственные имена, ТМ, термины без устоявшихся аналогов, цитаты и др. — по тексту закона |
| Ст. 4 закона № 168‑ФЗ | Ответственность | Административная ответственность по ст. 19.22 КоАП РФ (суммы и состав — по актуальной редакции) |

## Что обычно подпадает под действие в цифре

**Обычно подпадает:**

- корпоративные сайты и лендинги;
- мобильные приложения, доступные в РФ;
- публичные сообщества и каналы;
- чат-боты для неограниченного круга лиц;
- массовые рассылки и новостные ленты.

**Часто не рассматривается как «публичное пространство» в смысле этого блока:**

- личная переписка пользователь‑пользователь;
- закрытые группы без публичной коммерческой/информационной функции (спорные пограничные случаи — отдельно);
- сугубо внутренние админ‑интерфейсы (публичный продуктовый UI — отдельно).

## Практические требования к контенту

1. **Основной язык — русский** для информационных, рекламных и навигационных текстов.
2. **Иностранные слова** — при первом упоминании: `слово (пояснение на русском)`; в навигации предпочтительны русские формулировки.
3. **Исключения** — по списку закона: ТМ, термины без аналогов, цитаты и т. д.

### Интерфейс чат-бота (пример оформления)

| Элемент | Требование (практика) | Пример |
| ------- | --------------------- | ------ |
| Название бота | Иностранное допустимо как ТМ при наличии оснований | «ChatBot Pro» при регистрации как ТМ |
| Приветствие | На русском | «Здравствуйте! Я помогу…» |
| Кнопки | Русский или дублирование | «Каталог / Catalog» или только «Каталог» |
| Товары | Русское + уточнение при необходимости | «Смартфон Galaxy S24 (…произношение/пояснение…)» |
| Термины | Русский аналог или пояснение | «Вебхук (веб‑перехватчик)» |

## Санкции (ориентир по КоАП)

| Нарушение | Статья КоАП | Штрафы (ориентир) |
| --------- | ----------- | ----------------- |
| Иностранное слово без пояснения на русском | 19.22 ч. 1 | граждане 2 000–2 500; должностные 4 000–5 000; юрлица 40 000–50 000 |
| Повтор в течение года | 19.22 ч. 2 | выше по всем категориям |

## Чек-лист для цифрового контента

```
[ ] Проведён аудит всего публичного контента (сайт, бот, канал, рассылки)
[ ] Выявлены иностранные слова и выражения, требующие пояснения
[ ] Для каждого иностранного слова определено:
    [ ] Является ли оно частью зарегистрированного товарного знака
    [ ] Имеет ли устоявшийся русский аналог
    [ ] Является ли техническим термином без русского эквивалента
[ ] Внедрены правила оформления:
    [ ] При первом упоминании: иностранное слово + пояснение в скобках
    [ ] В интерфейсе: приоритет русскоязычным формулировкам
    [ ] В товарных названиях: русское название + иностранное при необходимости
[ ] Обновлены шаблоны ответов чат-бота на соответствие требованиям
[ ] Обучены контент-менеджеры и авторы правилам применения закона
[ ] Внедрена процедура проверки нового контента перед публикацией
[ ] Подготовлен глоссарий часто используемых иностранных терминов с русскими аналогами
```

## Правила для контент-менеджеров (сжатый регламент)

```
ПРАВИЛА ОФОРМЛЕНИЯ КОНТЕНТА (внутренний регламент)
в соответствии с требованиями законодательства о русском языке в публичных пространствах

1. Русский язык — основной для публичного контента.
2. Иностранные слова — с пояснением на русском при первом упоминании, если нет исключения.
3. Навигация — русскоязычные формулировки; дублирование допустимо.
4. Товарные знаки — без лишнего «перевода», если это ТМ.
5. Перед публикацией — чек-лист и глоссарий; спорные случаи — юрист/комплаенс.
6. Санкции — ст. 19.22 КоАП РФ (актуальная редакция).
```

## FAQ

**Нужно ли переводить английское имя бота?**  
Если это зарегистрированный ТМ — часто не требуется; иначе рекомендуется русское имя или дублирование вида `ChatHelper (Чат‑помощник)`.

**Нужно ли расшифровывать API, webhook?**  
Устоявшиеся технические термины могут иметь исключения; безопасная практика — пояснение при первом упоминании.

**Распространяется ли закон на личные сообщения поддержки?**  
Обычно нет для приватного диалога; публичные шаблоны/канал — да.

**Что при нарушении?**  
Риск предписания и штрафа по ст. 19.22 КоАП РФ для юрлиц (ориентир до 50 000 руб. за первое в классической схеме санкций — **проверять редакцию**).
