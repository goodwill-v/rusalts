## Релиз: безопасное встраивание виджета (VK/iframe) без утечек и блокировок

АЛТ‑эксперт изначально подготовлен к встраиванию в VK (iframe/WebView) с учётом требований безопасности браузера.

### Что сделали

- Настроили политику встраивания через CSP **`frame-ancestors`** (разрешён VK и домены проекта).
- Заложили корректный **CORS** для запросов из веб‑контекста VK.
- Перевели ссылки на статику на безопасный формат, чтобы избежать mixed‑content и проблем в HTTPS‑прокси.

### Почему это важно для бизнеса

- **Надёжная доставка интерфейса**: виджет не “падает” из‑за политики браузера.
- **Безопасность**: страница не открывается для встраивания кем угодно (только доверенные площадки).
- **Готовность к масштабированию каналов**: архитектура позволяет добавлять новые площадки без переделки ядра.
