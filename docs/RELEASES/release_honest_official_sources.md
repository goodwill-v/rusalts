## Релиз: “честные ответы” — только по базе знаний и официальным источникам

Мы усилили принцип доверия в АЛТ‑эксперт: если ответа нет в Базе знаний, система **не выдумывает**.

### Что сделали

- Добавили режим “official sources fallback”:
  - если в Базе знаний нет ответа, Backend ищет релевантные выдержки в перечне официальных ресурсов;
  - формирует ответ **строго по найденным выдержкам**;
  - обязательно прикладывает **ссылки на источник (URL)**.
- Если официальных подтверждений нет — отвечает честно:
  - **«Ответа пока нет в официальных источниках»**
  - и предлагает обратиться в поддержку для проработки решения.

### Почему это важно для бизнеса

- **Снижение юридических рисков**: никакой “галлюцинации” на комплаенсе и регуляторике.
- **Аудит и доверие**: каждое утверждение можно проверить по ссылке.
- **Единый стандарт качества**: одинаковые правила для чата, публикаций и обновлений.

### Что дальше

Дальше этот же принцип будет расширен:
- на более точную семантическую навигацию (эмбеддинги/векторное хранилище);
- на автоматический контроль “источник обязателен” для правовых статей и релизов.
