## 1. ОБЩИЕ СВЕДЕНИЯ

### 1.1. О проекте

АЛЬТЕРНАТИВА (АЛТ) — альтернативные легальные технологии. Проект помогает бизнесу, создателям контента и сообществам переходить из заблокированных мессенджеров и социальных сетей в отечественные цифровые платформы с соблюдением законодательства РФ.

### 1.2. Цель создания сайта

- Сформировать доверие к бренду АЛТ через демонстрацию экспертизы
- Обеспечить конверсию посетителей в заявки на консультации и услуги
- Предоставить актуальную информацию по правовым аспектам цифровизации
- Интегрировать ИИ-ассистента для первичной квалификации лидов
- Стать информационным хабом по теме миграции в отечественные платформы

### 1.3. Целевая аудитория

| Сегмент | Потребности | Ожидаемое действие на сайте |
|---------|------------|---------------------------|
| Предприниматели (ИП/ООО) | Перенести клиентов, настроить приём заказов, соблюсти 152-ФЗ | Запись на диагностику, скачивание чек-листов |
| Блогеры и инфлюенсеры | Сохранить монетизацию, легализовать контент | Консультация по миграции, подписка на новости |
| НКО и социальные сообщества | Легальная коммуникация, отчётность | Запрос на бесплатный аудит, участие в вебинарах |
| Юристы и комплаенс-специалисты | Актуальная информация по 152-ФЗ, 168-ФЗ | Подписка на обновления, запрос на партнёрство |
| ИТ-разработчики | Интеграция с МАХ/VK API, техническая документация | Доступ к базе знаний, контакт с тех. отделом |

## 2. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

### 2.1. Платформа и стек технологий

| Компонент | Требование | Обоснование |
|-----------|-----------|-------------|
| **CMS / Framework** | Next.js 14+ (React) или статический генератор (Astro) | Высокая скорость загрузки, SEO-оптимизация, масштабируемость |
| **Хостинг** | Российский провайдер с дата-центрами в РФ (Timeweb, Reg.ru, Yandex Cloud) | Соответствие 152-ФЗ, стабильность доступа |
| **SSL-сертификат** | Обязателен, выпущен доверенным УЦ в РФ | Безопасность, доверие пользователей, требования поисковых систем |
| **Адаптивность** | Мобильная версия (320px+), планшет (768px+), десктоп (1200px+) | Охват всех устройств, приоритет мобильному трафику |
| **Скорость загрузки** | Первая отрисовка контента (FCP) < 1.5 с, полная загрузка < 3 с | Удержание пользователей, фактор ранжирования |
| **Язык разметки** | HTML5, семантическая вёрстка | Доступность, SEO, поддержка скринридеров |

### 2.2. Требования к безопасности

- Защита от XSS, CSRF, SQL-инъекций
- Ограничение частоты запросов (rate limiting) для форм и API
- Регулярное резервное копирование (ежедневно, хранение 30 дней)
- Логирование действий администраторов
- Соответствие требованиям 152-ФЗ: политика конфиденциальности, согласие на обработку данных, локализация баз данных в РФ

### 2.3. Интеграции

| Интеграция | Назначение | Метод подключения |
|------------|-----------|------------------|
| ИИ-ассистент АЛТ-Эксперт | Консультации, квалификация лидов | REST API / WebSocket |
| CRM-система | Учёт заявок, управление клиентами | Веб-хуки / API |
| Email-рассылка | Информирование подписчиков | SMTP / API (UniSender, SendPulse) |

## 6. SEO И ПРОДВИЖЕНИЕ

### 6.1. Технические требования

- ЧПУ (человеко-понятные урлы): /uslugi/migraciya, /pravovaya-baza/152-fz
- Метатеги: title (до 60 символов), description (до 160 символов), keywords (опционально)
- Микроразметка: Schema.org для организации, статей, хлебных крошек
- Файлы: robots.txt, sitemap.xml, favicon.ico (32×32, 192×192)
- Open Graph и Twitter Cards для корректного отображения в соцсетях

### 6.2. Контентные требования

- Уникальность текстов: не менее 90% по Advego / Text.ru
- Ключевые слова: естественное вхождение, плотность 1–3%
- Заголовки: иерархия H1 → H2 → H3, один H1 на страницу
- Изображения: атрибуты alt, сжатие без потери качества (WebP + fallback)

### 6.3. Аналитика

- Яндекс.Метрика: цели на отправку форм, клики по кнопкам, время на сайте
- VK Pixel: отслеживание конверсий из рекламы ВКонтакте

- Планирование публикации по дате и времени
